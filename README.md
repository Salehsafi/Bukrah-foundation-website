# Bukrah foundation website
Case study of a Squarespace website built from scratch for Bukrah Foundation, focused on responsiveness, structure, and donation flow.

## Overview
This project is a case study of the Bukrah Foundation website, which I built from scratch on Squarespace after the previous website was fully removed.

The goal was to create a more polished, responsive, and user-friendly website that clearly presents the foundation’s mission, programs, and donation pathways.

## My Role
I was responsible for building and shaping the website experience from the ground up, with a focus on:
- responsive design
- clearer content structure
- stronger visual presentation
- improved usability
- better support for donation flow and engagement

## Project Focus
The website was designed to support a humanitarian organization through a clear and practical digital presence. The final structure highlights programs, donation calls to action, FAQ content, and communication channels in a more organized and accessible way.

## Tools & Platform
- Squarespace
- HTML/CSS customization
- JavaScript
- Responsive design
- UI/UX improvements

## Key Contributions
- Built the website from scratch after the previous site was removed
- Improved responsiveness across desktop and mobile views
- Created a cleaner and more structured layout
- Strengthened presentation of the organization’s mission and programs
- Helped support donation flow and community engagement

## Outcome
The result was a significantly improved website experience with stronger structure, clearer presentation, and a more polished responsive layout.
